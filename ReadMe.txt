(3) How to Deploy the Project:
	1. Download and Install Visual Studio Code.
	2. Run Visual Studio Code.
	3. Make sure to install MySQL in order to make the database.
	4. Open Folder provided by the name Source Code in Visual Studio Code.
	5. Install all the libraries and dependencies which Visual Studio Code is recommending in order to run the project.
	6. Your Internet Connectivity should be stable in order to get the Machine Learning results.
	7. Follow the port link after running the project in the terminal section of Visual Studio Code.
